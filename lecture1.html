Lecture 01

Set Theory & Operations

The mathematical language of collections — how to define, describe, compare, and combine sets. The foundation of logic, databases, and data science.

// Section 1

Set Theory

Definition

A set is an unordered collection of objects, called elements or members. A set is said to contain its elements.

We write a ∈ A to say "a belongs to A", and a ∉ A to say "a does not belong to A". Sets are named with uppercase letters; elements with lowercase.

Examples

People in a class: {Trisha, Tanvir, Tonmoy, Keya}

Mixed elements: {3, a, Potato}

Positive numbers ≤ 5: {1, 2, 3, 4, 5}

Key Properties

Order doesn't matter: {1,2,3} = {3,1,2}

Duplicates don't matter: {1,1,2,2} = {1,2}

// Section 1.2

Three Ways to Describe a Set

Method 1
Word Description

Describe the set in plain English.
Example: "The set of even counting numbers less than 10."

Method 2
Roster (Listing) Method

List all elements between curly braces.
Example: {2, 4, 6, 8}

Method 3
Set-Builder Notation

Characterize elements by a property they must satisfy.
Example: O = { x | x is an odd positive integer < 10 }
Or: B = { x ∈ ℤ⁺ | x is odd and x < 10 }
Read as: "B is the set of all x such that x is odd and x < 10."

Interval Notation

For continuous ranges of real numbers between a and b (where a < b):

[a, b]
Closed interval: a ≤ x ≤ b (both endpoints included)
(a, b)
Open interval: a < x < b (neither endpoint included)
[a, b)
Half-open: a ≤ x < b (left included, right excluded)
(a, b]
Half-open: a < x ≤ b (left excluded, right included)
// Section 1.3

Key Set Terminology

Universal Set (U)

Definition

U is the universal set — the set of all elements from which any given set is drawn. Every set exists within a universe.

Empty Set (∅)

Definition

A set with zero elements is the empty set, written ∅ = { }.

Important Distinction

∅ — empty set (0 elements)

{∅} — a set containing one element (the empty set). These are NOT the same!

Subset (⊆) and Proper Subset (⊂)

Definition — Subset

A ⊆ B if every element of A is also in B: ∀x (x ∈ A → x ∈ B)

Definition — Proper Subset

A ⊂ B if A ⊆ B and A ≠ B. There must exist some element in B not in A.

Examples

A = {2,4,6}, B = {1,2,3,4,5,6,7} → A ⊂ B and A ⊆ B ✓

A = {1,2,3,4}, B = {1,2,3,4} → A ⊆ B ✓ but A ⊄ B (they are equal)

Remember

For any set S:  ∅ ⊆ S  and  S ⊆ S  are always true.

Set Equality

Definition

A = B if and only if they have exactly the same elements: ∀x (x ∈ A ↔ x ∈ B)

Examples

{1,2,3} = {3,2,1}  →  True (order doesn't matter)

{1,2,3} = {3,3,2,1,2,1}  →  True (duplicates don't matter)

Cardinality (|S|)

Definition

The cardinality of a finite set S, written |S|, is the number of distinct elements in S.

Examples

A = {1,3,5,7,9}  →  |A| = 5

English alphabet  →  |S| = 26

|∅| = 0  and  |{∅}| = 1

Power Set P(S)

Definition

The power set P(S) is the set of all subsets of S. If |S| = n, then |P(S)| = 2ⁿ.

Example — P({0, 1, 2})

P({0,1,2}) = { ∅, {0}, {1}, {2}, {0,1}, {0,2}, {1,2}, {0,1,2} }

|S| = 3, so |P(S)| = 2³ = 8 subsets

Set Operations

Operations on sets A and B produce new sets. There are five key operations:

A ∪ B A ∩ B A − B A ⊕ B Ā

Union — A ∪ B
A ∪ B = { x | x ∈ A ∨ x ∈ B }
All elements that are in A, in B, or in both.

Union (∪)

Definition

A ∪ B contains elements that are in A or in B (or both).

A ∪ B = { x | x ∈ A ∨ x ∈ B }

Example

A = {1,2,5,7}, B = {3,4,5,6}

A ∪ B = {1, 2, 3, 4, 5, 6, 7}

Intersection (∩)

Definition

A ∩ B contains elements that are in both A and B.

A ∩ B = { x | x ∈ A ∧ x ∈ B }

Example

A = {1,2,5,7}, B = {3,4,5,6}

A ∩ B = {5}

Disjoint Sets

Two sets are disjoint if A ∩ B = ∅ (no elements in common).

Example: {a, b} and {3, 4} are disjoint.

Difference (−)

Definition

A − B contains elements in A that are not in B.

A − B = { x | x ∈ A ∧ x ∉ B }

Examples

A = {1,3,5}, B = {1,2,3} → A − B = {5}

A = {1,3,5,6}, B = {1,2,3,9,10} → A − B = {5, 6}

Note

A − B ≠ B − A in general. The difference operation is not commutative.

Symmetric Difference (⊕)

Definition

A ⊕ B contains elements in A or B, but not both.

A ⊕ B = { x | x ∈ A ∨ x ∈ B ∧ x ∉ (A ∩ B) }

Example

A = {1,2,5,7}, B = {3,4,5,6}

A ⊕ B = {1, 2, 3, 4, 6, 7}  (5 is excluded since it's in both)

Complement (Ā)

Definition

The complement of A (written Ā) is everything in U that is not in A.

Ā = { x ∈ U | x ∉ A } = U − A

Example

If U = {1,2,3,4,5} and A = {1,3,5}, then Ā = {2, 4}
