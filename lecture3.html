Solving Inequalities

Master the art of bounding — from linear inequalities and their manipulation rules, to quadratic sign analysis and absolute-value inequalities. Every solution is a set of numbers, expressed with precision.

// Section 1

What is an Inequality?

Sometimes a problem requires us to find all numbers satisfying a condition, not just one. An inequality is like an equation, except the equals sign is replaced by one of:

< ≤ > ≥
Definition

An inequality is a mathematical statement asserting that one expression is greater than, less than, greater than or equal to, or less than or equal to another expression.

Example

2x + 1 ≤ 3 is an inequality.

Its solution is the set of all x ≤ 1, i.e. the interval (-∞, 1].

Interval Notation Reminder

Closed
[a, b] ↔ a ≤ x ≤ b
Both endpoints included (filled dot ●)
Open
(a, b) ↔ a < x < b
Both endpoints excluded (open dot ○)
Half-Open Left
[a, b) ↔ a ≤ x < b
Left included, right excluded
Unbounded
[a, ∞) ↔ x ≥ a
Infinity is never included — always use ( )

Linear Inequalities

A linear inequality involves a first-degree polynomial. We solve it by isolating the variable, following six manipulation rules.

The Six Rules

The rules below hold for ≤. They apply equally to <, >, ≥. Rule 4 is the crucial one — multiplying/dividing by a negative flips the sign.

Rule 1 — Add
A ≤ B ⟺ A + C ≤ B + C
Adding the same value to both sides preserves the inequality.
Rule 2 — Subtract
A ≤ B ⟺ A − C ≤ B − C
Subtracting the same value preserves the inequality.
Rule 3 — Multiply (C > 0)
A ≤ B ⟺ CA ≤ CB
Multiplying by a positive number preserves the direction.
Rule 4 — Multiply (C < 0) ⚠️
A ≤ B ⟺ CA ≥ CB
Multiplying by a NEGATIVE flips the inequality sign!
Rule 5 — Reciprocal
If A,B > 0 and A ≤ B, then 1/A ≥ 1/B
Taking reciprocals of positive numbers flips the direction.
Rule 6 — Addition
If A ≤ B and C ≤ D, then A+C ≤ B+D
Two inequalities of the same direction can be added.

Worked Examples

Example 1 — Solve 2x + 3 > 11

Find the solution set.

// Step-by-step solution
⟹2x + 3 > 11Start
⟹2x > 11 − 3Rule 2: subtract 3
⟹2x > 8
⟹x > 4Rule 3: divide by 2 (positive)
S.S. = ]4, +∞[
Example 2 — Solve 3x − 5 ≤ 4
// Step-by-step solution
⟹3x − 5 ≤ 4
⟹3x ≤ 4 + 5Rule 1: add 5
⟹3x ≤ 9
⟹x ≤ 3Rule 3: divide by 3 (positive)
S.S. = ]−∞, 3]
Example 3 — Solve 2x + 1 ≤ 5x + 10
// Step-by-step solution
⟹2x + 1 ≤ 5x + 10
⟹2x − 5x ≤ 10 − 1Collect x terms on left
⟹−3x ≤ 9
⟹x ≥ −3Rule 4: divide by −3, FLIP sign ⚠️
S.S. = [−3, +∞[

Non-Linear (Quadratic) Inequalities

When the inequality involves x², we cannot simply divide both sides. Instead, we factorise and use a sign chart to determine where the expression is positive or negative.

General Strategy

Step 1: Move all terms to one side — get 0 on the right.

Step 2: Factorise the left side.

Step 3: Find the zeros (roots) of each factor.

Step 4: Build a sign chart — test each interval.

Step 5: Read off the solution set.

The Sign Chart Method

The sign chart tracks the sign (+/−) of each factor and their product across every interval defined by the roots.

Sign Rules for Products

(−) × (−) = (+)    (−) × (+) = (−)    (+) × (+) = (+)

Worked Example — x² + 3 ≤ 4x

Solve x² + 3 ≤ 4x
// Step 1: Move everything to one side
⟹x² − 4x + 3 ≤ 0
// Step 2: Factorise
⟹(x − 3)(x − 1) ≤ 0
// Step 3: Roots are x = 1 and x = 3

Build the sign chart over three intervals: (−∞, 1), (1, 3), (3, +∞)

Interval → (−∞, 1) (1, 3) (3, +∞)
Sign of (x − 1) − + +
Sign of (x − 3) − − +
Sign of (x−1)(x−3) + − ✓ +

We want (x−1)(x−3) ≤ 0, which means the product is negative or zero. This holds on [1, 3] (including endpoints where the product equals zero).

Solution

S.S. = [1, 3]

More Worked Examples

Example — Solve x² − 4x − 12 > 0
// Factorise: find two numbers that multiply to −12 and sum to −4
⟹(x − 6)(x + 2) > 0
// Roots: x = 6 and x = −2. Sign chart:
Interval →(−∞, −2)(−2, 6)(6, +∞)
Sign of (x + 2)−++
Sign of (x − 6)−−+
Sign of product+ ✓−+ ✓

We want the product positive, so we take the intervals where it is positive, excluding the roots (strict inequality).

Solution

S.S. = ]−∞, −2[ ∪ ]6, +∞[

Example — Solve x² − 6x + 8 ≤ 0
⟹(x − 2)(x − 4) ≤ 0
// Roots: x = 2 and x = 4
Interval →(−∞, 2)(2, 4)(4, +∞)
Sign of (x − 2)−++
Sign of (x − 4)−−+
Sign of product+− ✓+
Solution

S.S. = [2, 4]

Absolute Value Inequalities

The absolute value |x| measures the distance of x from zero:

|x| = x if x > 0,   −x if x ≤ 0

When solving inequalities involving |A|, we split into two cases depending on the direction of the inequality.

Properties of Absolute Value Inequalities

For a constant c > 0 and any algebraic expression A:

InequalityEquivalent FormType
|A| < c −c < A < c  (AND / simultaneous) A single interval
|A| ≤ c −c ≤ A ≤ c  (AND / simultaneous) A single interval
|A| > c A < −c   OR   A > c  (not simultaneous) Two separate intervals
|A| ≥ c A ≤ −c   OR   A ≥ c  (not simultaneous) Two separate intervals
Key Insight

< and ≤ → the solution is a single interval (AND condition).

> and ≥ → the solution is a union of two intervals (OR condition).

Worked Examples

Example 1 — Solve |x − 4| < 5
// |A| < c → −c < A < c
⟹−5 < x − 4 < 5
⟹−5 + 4 < x < 5 + 4Add 4 to all parts
⟹−1 < x < 9
S.S. = (−1, 9)
Example 2 — Solve |x + 2| > 8
// |A| > c → A < −c OR A > c
⟹x + 2 < −8   OR   x + 2 > 8
⟹x < −10   OR   x > 6
S.S. = (−∞, −10] ∪ [6, +∞)
Example 3 — Solve |3x + 4| < 2
// |A| < c → −c < A < c
⟹−2 < 3x + 4 < 2
⟹−6 < 3x < −2Subtract 4 from all parts
⟹−2 < x < −⅔Divide by 3 (positive — no flip)
S.S. = (−2, −⅔)
