Exponents & Algebra

From repeated multiplication to the laws that govern powers — plus algebraic expansion and equation-solving strategies used throughout calculus.

// Section 1

Index Notation

When a number is multiplied by itself repeatedly, we use exponent notation to write it compactly. Instead of writing 2 × 2 × 2 × 2 × 2, we write:

25
5 = exponent / power / index 2 = base
Definition

If n is a positive integer, then aⁿ is the product of n factors of a:

aⁿ = a × a × a × … × a  (n factors)

Evaluate

(-2)³ = -8  |  2⁴ = 16  |  (-1)⁵ = -1  |  (-1)⁴ = 1

(1/3)³ = 1/27  |  0³ = 0

// Section 2

Negative Bases

Base = −1
(-1)¹= -1
(-1)²= 1
(-1)³= -1
(-1)⁴= 1
Base = −2
(-2)¹= -2
(-2)²= 4
(-2)³= -8
(-2)⁴= 16
Key Rule

A negative base raised to an odd power → negative.

A negative base raised to an even power → positive.

Watch out for parentheses!

(-2)⁴ = 16  vs  -2⁴ = -1 × 2⁴ = -16

(-2)⁵ = -32  vs  -(−2)⁵ = -1 × (-32) = 32

Index Laws

Click any law card to see a worked example:

aᵐ × aⁿ = aᵐ⁺ⁿ
Same base: keep base, add exponents.
3⁵ × 3¹⁰ = 3¹⁵
2⁵ × 2³ = 2⁵⁺³ = 2⁸ = 256
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
Same base: keep base, subtract exponents.
3⁵ / 3¹⁰ = 3⁻⁵
2⁵ / 2³ = 2⁵⁻³ = 2² = 4
(aᵐ)ⁿ = aᵐˣⁿ

(3⁵)¹⁰ = 3⁵⁰
(2²)³ = 2²ˣ³ = 2⁶ = 32
(ab)ⁿ = aⁿbⁿ
Power of a product = product of powers.
(2×4)³ = 2³×4³ = 8×64 = 512
(3·5)¹⁰ = 3¹⁰·5¹⁰
(a/b)ⁿ = aⁿ/bⁿ
Power of a quotient = quotient of powers.
(3/5)¹⁰ = 3¹⁰/5¹⁰
(2/3)⁴ = 2⁴/3⁴ = 16/81
a⁰ = 1  (a≠0)
Any non-zero number raised to 0 is 1.
(-2)⁰ = 1
1000⁰ = 1    x⁰ = 1
a⁻ⁿ = 1/aⁿ
Negative exponent = reciprocal of positive.
2⁻³ = 1/2³ = 1/8
x⁻³ = 1/x³    1/x⁻² = x²

Special case of negative exponent with n=1.

Rational Indices (Radicals & Roots)

Definition

a^(1/n) = ⁿ√a (the nth root of a)

In general: a^(m/n) = ⁿ√(aᵐ) = (ⁿ√a)ᵐ

NotationEquivalentExample
a^(1/2)√a9^(1/2) = √9 = 3
a^(1/3)³√a8^(1/3) = ³√8 = 2
a^(1/n)ⁿ√a32^(1/5) = ⁵√32 = 2
a^(m/n)ⁿ√(aᵐ)8^(2/3) = (³√8)² = 4
a^(-1/2)1/√a4^(-1/2) = 1/2

Properties of nth Roots

Key Properties

1.  ⁿ√(ab) = ⁿ√a · ⁿ√b    e.g. (27×8)^(1/3) = 3×2 = 6

2.  ⁿ√(a/b) = ⁿ√a / ⁿ√b    e.g. (27/8)^(1/3) = 3/2

3.  ᵐ√(ⁿ√a) = ᵐⁿ√a    e.g. ³√(²√64) = ⁶√64

4.  ⁿ√(aⁿ) = a if n is odd    e.g. ³√((-3)³) = -3

5.  ⁿ√(aⁿ) = |a| if n is even    e.g. ²√((-3)²) = 3

Write as power of 2

³√2 = 2^(1/3)   |   1/√2 = 2^(-1/2)   |   ⁵√4 = (2²)^(1/5) = 2^(2/5)

Algebraic Expansion

Polynomials

Definition

A polynomial in x is an expression of the form:

aₙxⁿ + aₙ₋₁xⁿ⁻¹ + … + a₁x + a₀

where aₙ ≠ 0 and n is the degree. Each aₖxᵏ is a term; aₖ is the coefficient.

Calculate (add polynomials)

(2x³ + x + 5) + (x⁵ + x³ + 10) = x⁵ + 3x³ + x + 15

(x² − 1)(x + 1) = x³ + x² − x − 1

Expansion Laws

(a+b)(c+d) = ac+ad+bc+bd
FOIL method
(a+b)(a−b) = a²−b²
Difference of squares
(a+b)² = a²+2ab+b²
Perfect square (+)
(a−b)² = a²−2ab+b²
Perfect square (−)
Worked Example — Expand & Simplify

x^(-1/2) (x^(3/2) + 2x^(1/2) − 3x^(-1/2))

1

Distribute x^(-1/2) into each term:

= x^(-1/2 + 3/2) + 2x^(-1/2 + 1/2) − 3x^(-1/2 − 1/2)
2

Add the exponents:

= x¹ + 2x⁰ − 3x⁻¹
3

Simplify (x⁰ = 1, x⁻¹ = 1/x):

= x + 2 − 3/x

A linear equation in one variable is equivalent to: ax + b = 0

Quadratic Equations

Definition

A quadratic equation has the form ax² + bx + c = 0, where a ≠ 0.

Zero-Product Property

AB = 0 if and only if A = 0 or B = 0. This is the key to solving by factoring.

Solve by Factoring — Three Examples

1. x² + 2x = 24
x² + 2x − 24 = 0rearrange
(x − 4)(x + 6) = 0factor
S.S = {−6, 4}
2. x² − 7x + 12 = 0
(x − 4)(x − 3) = 0factor
S.S = {3, 4}
3. x² − 8x − 20 = 0
(x + 2)(x − 10) = 0factor
S.S = {−2, 10}

Equations with Absolute Value

Definition

|a| = a if a ≥ 0, and |a| = −a if a < 0.

To solve |A| = B, solve both A = B and −A = B.

Key Facts

|−a| = |a| = a   |   |ab| = |a|·|b|   |   |−5| = |5| = 5

Solve for x

1. |3x + 1| = 5
Case 1:  3x + 1 = 5→ x = 4/3
Case 2:  −(3x + 1) = 5→ −3x = 6 → x = −2
S.S = {−2, 4/3}
2. |x² + 2x| = 1
Case 1:  x² + 2x = 1
         x² + 2x − 1 = 0no rational roots
Case 2:  −(x² + 2x) = 1
         x² + 2x + 1 = 0 → (x+1)² = 0
S.S = {−1}
