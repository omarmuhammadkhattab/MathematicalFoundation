Odd and Even Functions

Symmetry is a key property of many functions. We classify functions by how they behave when we replace x with −x.

Even Functions

Definition — Even

A function f is even if:

f(−x) = f(x)    for all x in the domain

Geometric meaning: the graph is symmetric with respect to the y-axis.

Classic Example

f(x) = x² is even because: f(−x) = (−x)² = x² = f(x) ✓

// y = x² — even function, y-axis symmetry. f(−x) = f(x)

Odd Functions

Definition — Odd

A function f is odd if:

f(−x) = −f(x)    for all x in the domain

Geometric meaning: the graph is symmetric about the origin.

Classic Example

f(x) = x³ is odd because: f(−x) = (−x)³ = −x³ = −f(x) ✓

// y = x³ — odd function, origin symmetry. f(−x) = −f(x)

Example — Classify Each Function

Problem

Classify each: (a) f(x) = x⁵ + x   (b) g(x) = 1 − x⁴   (c) h(x) = 2x − x²

(a) f(x) = x⁵ + x
f(−x) = (−x)⁵ + (−x)
= −x⁵ − x
= −(x⁵ + x)
= −f(x)
✓ ODD function
(b) g(x) = 1 − x⁴
g(−x) = 1 − (−x)⁴
= 1 − x⁴
= g(x)
✓ EVEN function
(c) h(x) = 2x − x²
h(−x) = 2(−x) − (−x)² = −2x − x²
Is h(−x) = h(x)?   −2x − x² ≠ 2x − x²   ✗
Is h(−x) = −h(x)?   −2x − x² ≠ −(2x − x²) = −2x + x²   ✗
⚠ NEITHER even nor odd
// Comparing symmetry: even (y-axis), odd (origin), neither (no symmetry)
// Section 2

One-to-One Functions (1-1)

Definition

A function f is one-to-one (injective) if:

f(x₁) = f(x₂)   implies   x₁ = x₂

In other words: no two different inputs produce the same output.

Equivalently: if x₁ ≠ x₂ then f(x₁) ≠ f(x₂). Each y-value is hit by at most one x-value.

Horizontal Line Test

The Test

A function is one-to-one if and only if every horizontal line intersects its graph at most once.

// f(x) = 2x + 3 — ONE-TO-ONE ✓
// f(x) = x² + 3 — NOT one-to-one ✗

Solved Examples

Example A
f(x) = 2x + 3
Assume f(x₁) = f(x₂)
2x₁ + 3 = 2x₂ + 3
2x₁ = 2x₂
x₁ = x₂
✓ f is one-to-one
Example B
f(x) = x² + 3
f(x₁) = f(x₂)
x₁² + 3 = x₂² + 3
x₁² = x₂²
x₁ = ±x₂
✗ NOT 1-1 (e.g. f(1)=f(−1)=4)
Restricted Domain Fix

y = x² + 3 is not 1-1 on ℝ, but is 1-1 if we restrict to x ≥ 0 (or x ≤ 0).

On [0, ∞): x₁² = x₂² with both ≥ 0 forces x₁ = x₂. ✓

Example C
f(x) = ³√(x + 1)
f(x₁) = f(x₂) → ³√(x₁+1) = ³√(x₂+1)
Cube both sides: x₁+1 = x₂+1 → x₁ = x₂
✓ f is one-to-one
// Section 3

Inverse Functions

Definition

If f is a one-to-one function, then its inverse f⁻¹ is the function that reverses f:

f⁻¹(y) = x   ⟺   f(x) = y

Key Properties

(f ∘ f⁻¹)(x) = x for all x in the domain of f⁻¹

(f⁻¹ ∘ f)(x) = x for all x in the domain of f

The domain of f⁻¹ = Range of f, and Range of f⁻¹ = Domain of f

How to Find an Inverse

  1. Verify f is one-to-one (or restrict domain)
  2. Write y = f(x)
  3. Solve for x in terms of y
  4. Replace y with x to get f⁻¹(x)

Example 1 — f(x) = 5x + 2

Finding f⁻¹

y = 5x + 2  →  y − 2 = 5x  →  x = (y − 2)/5

f⁻¹(x) = (x − 2) / 5

// f(x) = 5x+2 and f⁻¹(x) = (x−2)/5 — reflections across y = x

Verifying with Composition

Verification for f(x) = 5x + 2, f⁻¹(x) = (x−2)/5

(f ∘ f⁻¹)(x) = f(f⁻¹(x)) = f((x−2)/5) = 5·(x−2)/5 + 2 = (x−2) + 2 = x ✓

(f⁻¹ ∘ f)(x) = f⁻¹(f(x)) = f⁻¹(5x+2) = (5x+2−2)/5 = 5x/5 = x ✓

Example 2 — Exponential & Logarithm are Inverses

Classic Pair

f(x) = eˣ  and  f⁻¹(x) = ln x   are inverses of each other.

Domain of eˣ = ℝ    Range of eˣ = (0, ∞)

Domain of ln x = (0, ∞)    Range of ln x = ℝ

(f ∘ f⁻¹)(x) = e^(ln x) = x ✓    (f⁻¹ ∘ f)(x) = ln(eˣ) = x ✓

// f(x)=eˣ and f⁻¹(x)=ln x — reflections across y=x

More Inverse Examples

f(x) = √(4x + 3)
y = √(4x+3)
y² = 4x + 3
4x = y² − 3
x = (y² − 3)/4
f⁻¹(x) = (x² − 3) / 4  , x ≥ 0
f(x) = e^(x−2)
y = e^(x−2)
ln y = x − 2
x = ln y + 2
f⁻¹(x) = ln(x) + 2
f(x) = 1/x
y = 1/x
x = 1/y
f⁻¹(x) = 1/x  (self-inverse!)
f(x) = ln(2x + 5)
y = ln(2x+5)
eʸ = 2x + 5
x = (eʸ − 5)/2
f⁻¹(x) = (eˣ − 5) / 2
