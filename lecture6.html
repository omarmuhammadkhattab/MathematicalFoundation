Absolute Value Function — Extended

We study two important variations of the absolute value function, analysing their domain, range, and graphs.

Example 1 — f(x) = |x| / x

Function

f(x) = |x| / x

We rewrite using the definition of absolute value:

|x| =
x
if x > 0
−x
if x < 0

So f(x) = |x|/x simplifies to:

f(x) =
x/x = 1
if x > 0
−x/x = −1
if x < 0
Result

Domain = ℝ − {0}    (x = 0 makes denominator zero)

Range = {−1, 1}

// Graph of f(x) = |x|/x — two horizontal rays, hole at origin

Example 2 — f(x) = |x − 2|

Function

f(x) = |x − 2|

f(x) =
x − 2
if x − 2 ≥ 0  (x ≥ 2)
−(x − 2) = 2 − x
if x − 2 < 0  (x < 2)
Result

Domain = ℝ    Range = [0, ∞)

The vertex (minimum point) is at (2, 0) — the function equals zero there and opens upward.

Quadratic Functions

Definition

A quadratic function is a polynomial of degree 2:

f(x) = ax² + bx + c    (a ≠ 0)

The graph of a quadratic function is called a parabola. Its direction depends on the sign of a:

a > 0
Opens Upward ∪
Minimum value at vertex
Range = [k, ∞) where k is vertex y
a < 0
Opens Downward ∩
Maximum value at vertex
Range = (−∞, k] where k is vertex y

Vertex Formula

Vertex Coordinates

x-coordinate of vertex = −b / (2a)

y-coordinate = f(−b / 2a)

If a > 0: minimum value of f is f(h) = k at vertex (h, k)

If a < 0: maximum value of f is f(h) = k at vertex (h, k)

Domain & Range — Six Examples

All quadratic functions have Domain = ℝ. The range depends on the vertex and direction.

// y = x² − 3   Vertex (0, −3)
// y = −x² + 5   Vertex (0, 5)

// Section 2 — Technique

Completing the Square

Completing the square converts ax² + bx + c into vertex form a(x − h)² + k, making the vertex immediately visible.

Formula

x² + bx + c = (x + b/2)² − (b/2)² + c

Example — Simplify y = x² + 6x + 3

Solution

y = x² + 6x + 3

= (x + 6/2)² − (6/2)² + 3    ← complete the square

= (x + 3)² − 9 + 3

= (x + 3)² − 6    Vertex: (−3, −6)

Full Graphing Example — y = x² − 7x + 12

We use two methods to graph and find the vertex.

Method 1 — Complete the Square

y = x² − 7x + 12

= (x − 7/2)² − (7/2)² + 12

= (x − 3.5)² − 12.25 + 12

= (x − 3.5)² − 0.25    Vertex: (3.5, −0.25)

Method 2 — Systematic (4 Steps)

① a = 1 > 0 → parabola opens upward

② x-intercepts: set y = 0 → x² − 7x + 12 = 0 → (x−3)(x−4) = 0 → x = 3 or x = 4

③ y-intercept: set x = 0 → y = 12

④ Vertex x = −b/2a = 7/2 = 3.5; y = (3.5)² − 7(3.5) + 12 = −0.25

Vertex: (3.5, −0.25)

// Graph of y = x² − 7x + 12 — vertex (3.5, −0.25), roots at x=3 and x=4

Square Root Functions

Square root functions require the expression under the radical to be non-negative, which restricts the domain.

Key Rule

For f(x) = √(expression), require expression ≥ 0

// y = √x
// y = √x + 2
// y = √x − 3

Example — y = √(x − 4)

Finding the Domain

Require: x − 4 ≥ 0  →  x ≥ 4

Domain = [4, ∞)    Range = [0, ∞)

The graph is the standard √x curve shifted 4 units to the right.

// y = √(x − 4) — starts at (4, 0), shifted right

Cubic Functions

Standard Cubic

f(x) = x³

Domain = ℝ    Range = ℝ

The cubic function is an odd function — it is symmetric about the origin. It increases for all x and passes through (0,0), (1,1), (−1,−1).

// Graph of y = x³ — origin symmetry, Domain = ℝ, Range = ℝ
// Section 5

Exponential Functions

Definition

Exponential functions have the variable in the exponent. The base e ≈ 2.718 (Euler's number) is the most natural base.

f(x) = eˣ    Domain = ℝ, Range = (0, ∞)

f(x) = e⁻ˣ    Domain = ℝ, Range = (0, ∞)

Both functions always stay strictly positive. The x-axis is a horizontal asymptote.

// y = eˣ — increasing, asymptote y=0
// y = e⁻ˣ — decreasing, asymptote y=0
// Section 6

Logarithm Function

Natural Logarithm

f(x) = ln x

Domain = (0, ∞)    (x must be strictly positive)

Range = ℝ

The y-axis (x = 0) is a vertical asymptote. The function passes through (1, 0) and (e, 1).

The natural log is the inverse of the exponential function eˣ. Their graphs are reflections of each other across the line y = x.

// Graph of y = ln x — Domain (0,∞), Range ℝ, passes through (1, 0)

Operations on Functions

Given functions f and g with domains Df and Dg, we define new functions using the four basic operations.

OperationFormulaDomain
Sum(f + g)(x) = f(x) + g(x)Df ∩ Dg
Difference(f − g)(x) = f(x) − g(x)Df ∩ Dg
Product(f · g)(x) = f(x) · g(x)Df ∩ Dg
Quotient(f/g)(x) = f(x)/g(x)Df ∩ Dg − {x : g(x) = 0}

Example 1 — f(x) = √(x+1), g(x) = √(4−x²)

Step 1 — Find each domain

Df: x + 1 ≥ 0 → x ≥ −1  →  Df = [−1, ∞)

Dg: 4 − x² ≥ 0 → x² ≤ 4 → |x| ≤ 2  →  Dg = [−2, 2]

Df ∩ Dg = [−1, 2]

Results

(f + g)(x) = √(x+1) + √(4−x²)    Domain: [−1, 2]

(f − g)(x) = √(x+1) − √(4−x²)    Domain: [−1, 2]

(f · g)(x) = √[(x+1)(4−x²)]    Domain: [−1, 2]

(f/g)(x) = √[(x+1)/(4−x²)]    Domain: [−1, 2)  (exclude x=2 where g=0)

Example 2 — f(x) = √x, g(x) = 2√x

Solution

Df = [0, ∞), Dg = [0, ∞)  →  Df ∩ Dg = [0, ∞)

(f · g)(x) = √x · 2√x = 2x    Domain: [0, ∞)

Note: even though 2x is defined for all ℝ, we restrict to [0, ∞) because of the original functions.

Composite Functions

Definition

(f ∘ g)(x) = f(g(x))  — apply g first, then feed result into f

(g ∘ f)(x) = g(f(x))  — apply f first, then feed result into g

⚠️ Composition is NOT commutative: f∘g ≠ g∘f in general

// Flow diagram of (f ∘ g)(x)

Example 1 — f(x) = √x, g(x) = x² + 1

Solution

(f ∘ g)(x) = f(g(x)) = f(x² + 1) = √(x² + 1)

(g ∘ f)(x) = g(f(x)) = g(√x) = (√x)² + 1 = x + 1

Note: f∘g ≠ g∘f

Example 2 — Find g(x) given f and f∘g

Problem

Given f(x) = 2x + 1 and (f ∘ g)(x) = x³, find g(x).

Solution

(f ∘ g)(x) = f(g(x)) = x³

2g(x) + 1 = x³

2g(x) = x³ − 1

g(x) = ½(x³ − 1)
