Functions & Logarithms

The heart of analysis — defining functions, evaluating them, reading graphs with the vertical line test, finding domain and range, and mastering the logarithm as an inverse of exponentiation.

// Section 1

Definition of a Function

Definition

A function f is a rule that assigns to each element x in a set A exactly one element, called f(x), in a set B.

A is the domain (inputs), and the set of all outputs f(x) is the range.

Input
x
→
Function
f
→
Output
f(x)
Key Point — "Exactly One"

Every input x must map to exactly one output. One input can't produce two different outputs — that would make it not a function.

Is it a Function?

Check whether any input value (first coordinate) appears more than once. If it does with different outputs — it is not a function.

Relation R
{(9,10), (−5,−2), (2,−1), (3,−9)}
✓ Function — each input appears exactly once.
Relation S
{(6,a), (8,f), (6,b), (−2,p)}
✗ Not a function — input 6 maps to both a and b.
Relation T
{(z,7), (y,−5), (r,7), (z,0), (k,0)}
✗ Not a function — z maps to both 7 and 0.
Note on Outputs
Two different inputs can share the same output — that's fine.
✓ r and z both map to 7 in T — that's allowed (if z were consistent).

Evaluating Functions

In f(x), the variable x is a placeholder. To evaluate f at a number, substitute that number everywhere x appears.

f(x) = 2x − 5
Find f(3):
f(3) = 2(3) − 5
f(3) = 6 − 5
f(3) = 1 → (3, 1)
f(x) = 4x − 1
Find f(2):
f(2) = 4(2) − 1
f(2) = 8 − 1
f(2) = 7 → (2, 7)
g(x) = x² − 2x
Find g(−3):
g(−3) = (−3)² − 2(−3)
g(−3) = 9 + 6
g(−3) = 15 → (−3, 15)

The Difference Quotient

For a function f and h ≠ 0, the difference quotient is:

[f(a + h) − f(a)] / h

This quantity measures the average rate of change of f between x = a and x = a + h. It is fundamental to differential calculus.

Example — f(x) = 2x² − 5x + 1, evaluate [f(a+h) − f(a)] / h
f(a + h) = 2(a + h)² − 5(a + h) + 1
= 2(a² + 2ah + h²) − 5a − 5h + 1
= 2a² + 4ah + 2h² − 5a − 5h + 1
f(a+h) − f(a) = (2a² + 4ah + 2h² − 5a − 5h + 1) − (2a² − 5a + 1)
= 4ah + 2h² − 5h Cancel matching terms
[f(a+h)−f(a)] / h = (4ah + 2h² − 5h) / h = 4a + 2h − 5
// Section 3

Graphing Functions

To graph a function f, plot the points (x, f(x)) in the coordinate plane for various values of x, then connect them.

Key Graphs to Know

f(x) = x² — Parabola opening upward, vertex at origin.

g(x) = x³ — Cubic curve through origin, symmetric about origin.

h(x) = √x — Square root curve, domain x ≥ 0, starts at origin.

f(x) = 2 + 5x — Straight line, slope 5, y-intercept 2.

The Vertical Line Test

The Vertical Line Test

A curve in the coordinate plane is the graph of a function if and only if no vertical line intersects the curve more than once.

Reason: if a vertical line x = a hits two points, then input a produces two outputs — violating the definition of a function.

Interactive Vertical Line Test
Select a curve, then drag the vertical line to test it.
✓ Function — no vertical line intersects the curve more than once.

Domain & Range

Every function has a domain (the permitted inputs) and a range (the resulting outputs).

Natural Domain

Natural Domain

If no domain is specified, the natural domain is the largest set of real numbers for which the expression is defined.

Two main restrictions to watch for:

Division by zero
A denominator must never equal zero.
f(x) = 1/(x−3) → exclude x = 3
Domain: (−∞,3) ∪ (3,∞)
Square roots (even)
The radicand (expression under √) must be ≥ 0.
f(x) = √(x−2) → x−2 ≥ 0 → x ≥ 2
Domain: [2, ∞)
Logarithms
The argument of log must be strictly positive.
f(x) = log(x+1) → x+1 > 0 → x > −1
Domain: (−1, ∞)
Polynomials
No restrictions — all real numbers allowed.
f(x) = x³ − 2x + 1
Domain: (−∞, ∞) = ℝ

Domain & Range from Graphs

From a graph: scan horizontally for domain (how far left/right the curve extends) and vertically for range (how far up/down).

Reading Interval Notation from Graphs

A closed dot ● → endpoint is included → use [ or ]

An open dot ○ → endpoint is excluded → use ( or )

An arrow → → curve extends to ±∞ → use (−∞, ...) or (..., ∞)

Example

A graph has a closed dot at x = −3 (left end) and an open dot at x = 5 (right end), reaching as low as y = −4 and as high as y = 2.

Domain: [−3, 5)    Range: [−4, 2]

Logarithmic Functions

Definition

Logarithmic Function with Base a

For a > 0 and a ≠ 1:   y = loga x   if and only if   x = aʸ

Domain: 0 < x < ∞     Range: −∞ < y < ∞

The logarithm answers the question: "To what power must I raise a to get x?"

Special Cases

Natural log:   ln x = loge x   where e ≈ 2.718 (Euler's number)

Common log:   log x = log10 x

ln e = 1    ln 1 = 0    loga a = 1    loga 1 = 0

Examples

log10 100 = 2   because 10² = 100

log2 8 = 3    because 2³ = 8

log2 16 = 4   because 2⁴ = 16

log5 25 = 2   because 5² = 25

The Seven Logarithm Rules

These rules work for any valid base a. They follow directly from the laws of exponents.

Rule 1 — Product
log(x · y) = log x + log y
Log of a product = sum of logs
Rule 2 — Quotient
log(x/y) = log x − log y
Log of a quotient = difference of logs
Rule 3 — Power
log xⁿ = n · log x
Exponent comes down as a multiplier
Rule 4 — Base
loga a = 1
Log base a of a itself is always 1
Rule 5 — Exponent
loga aˣ = x
Inverse property — cancels the base
Rule 6 — Reciprocal
loga(1/x) = −loga(x)
Log of a reciprocal = negative log
Rule 7 — One
loga 1 = 0
Log of 1 is always 0, any base
Change of Base
logb x = ln x / ln b
Convert any base to natural log

Solving Logarithmic Equations

Convert between logarithmic and exponential form:   loga x = y ↔ x = aʸ

Example 1 — Solve log₃(4x − 7) = 2
// Convert to exponential form: log_a(B) = C → B = a^C
⟹4x − 7 = 3²
⟹4x − 7 = 9
⟹4x = 16
⟹x = 4
x = 4
Example 2 — Solve logx 64 = 2
// The base is unknown. Convert: x² = 64
⟹x² = 64
⟹x = ±√64 = ±8
⟹x = 8x = −8 rejected: log base must be positive
x = 8
Example 3 — Solve log x = 3 log 2 + 2 log 3
// Use Rule 3 (power) then Rule 1 (product)
⟹log x = log 2³ + log 3²Rule 3: n·log a = log aⁿ
⟹log x = log 8 + log 9
⟹log x = log(8 × 9)Rule 1: sum → product
⟹log x = log 72
⟹x = 72Logs equal → arguments equal
x = 72

Simplifying Logarithmic Expressions

Example 1 — Simplify log(x³ · √x)
⟹log(x³ · x^(1/2))Write √x = x^(1/2)
⟹log(x^(7/2))Multiply: x³ · x^(1/2) = x^(7/2)
⟹(7/2) log xRule 3: power comes down
(7/2) log x
Example 2 — Simplify log(x⁵ / x²)
⟹log x⁵ − log x²Rule 2: quotient → difference
⟹5 log x − 2 log xRule 3: power → multiplier
3 log x
Example 3 — Simplify ln[ x⁵ / (1+x)¹⁸ ]
⟹ln x⁵ − ln(1+x)¹⁸Rule 2: quotient → difference
⟹5 ln x − 18 ln(1+x)Rule 3: powers come down
5 ln x − 18 ln(1 + x)
