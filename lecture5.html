// Section 1

Logarithm Simplification — Review

Before extending our work on domains, we review and consolidate logarithm simplification using the three core log rules.

Core Rules Recap

Product: log(AB) = log A + log B

Quotient: log(A/B) = log A − log B

Power: log(Aⁿ) = n · log A

Worked Examples

Example 1

Simplify log(x³ · √x)

= log(x³ · x^(1/2)) = log(x^(3 + 1/2)) = log(x^(7/2)) = (7/2) log x
Example 2

Simplify log(x⁵ / x²)

= log(x^(5-2)) = log(x³) = 3 log x
Example 3

Simplify ln( x⁵ / (1+x)¹⁸ )

= ln(x⁵) − ln((1+x)¹⁸) = 5 ln x − 18 ln(1+x)

Calculating Domain Using Algebra

The domain of a function is the set of all input values x for which the function is defined — i.e., produces a real output. The key question to ask is always:

The Guiding Question

For what values of x does f(x) make sense?

Three Main Problem Types

There are three situations that restrict the domain. You must check for each one present in a function.

Situation Function Form Restriction Reason
Even Root y = √f(x) f(x) ≥ 0 Square roots of negatives are not real
Denominator y = 1/f(x) f(x) ≠ 0 → ℝ − {x : f(x)=0} Division by zero is undefined
Logarithm y = ln f(x) f(x) > 0 Log of zero or negatives is undefined
Odd Roots — No Restriction

Odd roots such as ∛f(x) can accept any real number, so they impose no domain restriction. The domain is all of ℝ.

Set C — Rational and Logarithmic Domains

Example
f(x) = 1/(x² − 6x + 8)
Denominator ≠ 0

f(x) = ln x
ln requires x > 0

Example
f(x) = ln(x − 12)
x − 12 > 0
x > 12

Example
f(x) = ln(10 − x)
10 − x > 0

Require x² − 8x + 15 > 0. Factor: (x−3)(x−5) > 0.

Region
(−∞, 3) (3, 5) (5, ∞)
Sign
+ ✓ − ✗ + ✓

Set D — Mixed Conditions

Example
f(x) = ∛(x² + 4x − 2)
Odd root — no restriction
D_f = ℝ
Example
f(x) = √(x−2) / (x−5)
Even root: x − 2 ≥ 0 → x ≥ 2
Denominator: x − 5 ≠ 0 → x ≠ 5
D_f = [2, ∞) − {5}
Example
f(x) = (x²+1) / √(x−10)
Denominator contains even root:
x − 10 > 0 (strict, since it's in denominator)
x > 10
D_f = (10, ∞)
Key Distinction

When a square root appears in a numerator, the condition is ≥ 0 (we allow the root to equal zero). When it appears in a denominator, the condition is > 0 (zero in the denominator is undefined).

Piecewise-Defined Functions

A piecewise function uses different formulas on different parts of its domain. Each formula applies only within its specified interval.

Definition

A function f is piecewise-defined if it is described by two or more formulas, each valid for a different subset (piece) of the domain.

Example 1 — Classic Three-Piece Function

f(x) =
−x
if x < 0
x²
if 0 ≤ x ≤ 1
1
if x > 1

The graph uses the branch y = −x for negative inputs, y = x² on [0, 1], and the constant y = 1 for inputs beyond 1.

// Graph of f(x)

Example 2 — Evaluating a Piecewise Function

f(x) =
1
if x ≥ 0
2
if x < 0
Solution

f(4) = 1  (since 4 ≥ 0)    f(10) = 1  (since 10 ≥ 0)

f(−3) = 2  (since −3 < 0)    f(−15) = 2  (since −15 < 0)

Domain: ℝ    Range: {1, 2}

Example 3 — Real-World Piecewise (Cost Function)

C(x) =
39
if 0 ≤ x ≤ 2
39 + 15(x − 2)
if x > 2
Solution

C(0.5): since 0.5 ≤ 2, use first piece → C(0.5) = 39

C(2): since 2 ≤ 2, use first piece → C(2) = 39

C(4): since 4 > 2, use second piece → C(4) = 39 + 15(4 − 2) = 39 + 30 = 69

// Section 3 — Special Case

Absolute Value as a Piecewise Function

The absolute value function is a classic piecewise function with domain ℝ and range [0, ∞).

Definition

|a| = a  if a ≥ 0   ,   |a| = −a  if a < 0

|x| =
x
if x ≥ 0
−x
if x < 0
Numerical Examples

|3| = 3    |−3| = −(−3) = 3    |0| = 0

|3 − π| = −(3 − π) = π − 3    (since 3 < π, so 3 − π < 0)

// Graph of y = |x| — V-shape, Domain ℝ, Range [0, ∞)
Domain and Range

Domain: (−∞, ∞) = ℝ    Range: [0, ∞)

The output is never negative — absolute value always returns a non-negative result.

Linear Functions

A function is linear if it can be written in the form f(x) = ax + b, where a is the slope and b is the y-intercept. Its graph is a straight line.

Standard Form f(x) = ax + b

a = slope (rate of change)    b = y-intercept (value when x = 0)

Constant Function — Special Case

When slope a = 0, the function f(x) = b is called a constant function. Its graph is a horizontal line. Domain = ℝ, Range = {b}.

Graphical Behaviour by Sign

// Effect of slope a on the line f(x) = ax + b
Domain and Range of Any Linear Function

For any non-constant linear function f(x) = ax + b with a ≠ 0: Domain = ℝ and Range = ℝ.

Reading the Slope

A slope of a = 3 means: for every 1 unit increase in x, the output f(x) increases by 3 units. A negative slope means the function is decreasing.
